# Face Recognition System 👁️ 

Интеллектуальная система распознавания лиц в реальном времени. Проект создан в рамках практики по ООП и демонстрирует навыки работы с компьютерным зрением и многопоточностью.

## 🔥 Ключевые фишки
* **Распознавание в реальном времени:** Плавный захват видео с веб-камеры благодаря выносу обработки в отдельный поток (`threading`).
* **Гибкость:** Поддержка обработки как живого видео, так и статичных изображений или видеофайлов.
* **База лиц:** Система автоматически подгружает эталоны из папки `Base`.
* **Оптимизация:** Использование алгоритмов сжатия кадра для стабильной работы нейросети.

## 🛠 Стек технологий
* **Language:** Python 3
* **AI/ML:** `face_recognition` (state-of-the-art модель распознавания лиц)
* **Computer Vision:** `OpenCV`
* **UI:** `Tkinter` (Custom GUI)
* **Threading:** Многопоточная обработка видеопотока

## 🚀 Быстрый старт
1. Установите необходимые библиотеки:
   `pip install -r requirements.txt`
2. Добавьте фото в папку `Base` (имя файла станет именем в системе).
3. Запустите проект:
   `python face-recognition-Tkinter.py`
